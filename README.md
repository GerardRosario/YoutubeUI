# YoutubeUI

A UI Copy of the Youtube Video Viewing/Sharing Website.

## Technology / Tools
- React
- JavaScript
- HtML
- CSS
- VS Studio Code

## Features
- Light Mode / Dark Mode
- In-App Video Viewing
